While

Estruturas de repetição servem para executar o mesmo bloco de código várias vezes.

O While serve para criar um laço de repetição que será executado enquanto uma determinada condição for verdadeira.

Atenção: é importante que sempre exista uma forma da condição ser falsa, ou então o bloco ficará se repetindo para sempre;
ENQUANTO condição FAÇA algo várias vezes;
